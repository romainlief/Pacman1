# Pacman1

## Description

This is a Python implementation of the classic Pacman game, designed for educational purposes. It includes various search algorithms to navigate Pacman through mazes, collect food dots, and avoid ghosts. Link to the CS188 course: [Pacman 1](https://inst.eecs.berkeley.edu/~cs188/fa25/projects/proj1/#q1-3-pts-finding-a-fixed-food-dot-using-depth-first-search)

## Dependencies

- Python 3.x
- Tkinter

## Usage

To run the Pacman game from the root directory, execute the following command in your terminal:

```bash
cd search && python pacman.py
```
